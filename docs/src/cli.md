# CLI Reference

## `validate`

Validates a BCIF file.

```bash
open-bcif validate <INPUT>
```

## `split`

Splits a BCIF file.

```bash
open-bcif split <INPUT> [--output-dir <DIR>]
```

## `merge`

Merges BCIF files.

```bash
open-bcif merge <INPUTS>... --output <OUTPUT>
```

## `convert`

Converts BCIF files.

```bash
open-bcif convert <INPUT> --output <OUTPUT> --format <FORMAT>
```
